# Курсовая по ПИС
## Команды для проекта

| Команда                   | Действия                                         |
| :------------------------ | :----------------------------------------------- |
| `npm install`             | Установить зависимости                           |
| `npm dev`                 | Запустить локальный сервер `localhost:4321`      |
| `npm build`               | Забилдить проект `./dist/`                       |
| `npm preview`             | Посмотреть билд проекта локально                 |
